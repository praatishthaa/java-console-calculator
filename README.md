# Java Console Calculator

This is a console-based calculator implemented in Java as part of a Java Developer Internship task.

## Objective
To create a simple calculator that performs basic arithmetic operations using Java console I/O.

## Features
- Performs addition, subtraction, multiplication, and division
- Accepts user input using the Scanner class
- Executes operations inside a loop until the user exits
- Includes separate methods for each operation
- Handles divide-by-zero cases with appropriate messaging

## Technologies Used
- Java SE (JDK 17 or later)
- Any standard code editor or IDE (VS Code, IntelliJ)
- Terminal or Command Prompt

## How to Run
1. Compile the Java file:
   javac ConsoleCalculator.java
Run the program:
java ConsoleCalculator
Follow on-screen prompts to perform calculations. Type exit to close the program.
